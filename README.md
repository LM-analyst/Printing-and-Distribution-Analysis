# Printing-and-Distribution-Analysis
This repository is to showcase my analysis into journal print and distribution locations.

# Business Context
Following EUDR guidance, I was required to undertake analysis of where journals printed in different locations were distributed to, in order to identify opportunities to relocate printing with the aim of reducing environmental and financial costs. To achieve this, I collected data on the print copies of each journal and their distribution location. The data was cleansed extensively and combined in Python before being loaded into Power BI to visualise results.

# Business Impact
Producing the dashboard improved understanding of where journals are printed and distributed to, thereby identifying opportunities to relocate the printing of some journals to alternative locations in order to improve efficiency and reduce costs. The analysis also provided insights into the print quantities of each journal and their customer base.

# Tools Used
- Excel
- Power BI desktop
- Python (Pandas, os)
